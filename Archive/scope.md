# Purpose

This project is a personal introduction webpage for CMPA 3301 that will show I have gained the basic knowledge of HTML structure.

## Description

This webpage will introduce who I am, what I am studying, and where I am headed. This page will include my name, a short introduction, a list of hobbies or interests, and a link to my GitHub profile. I will use the required semantic tags: <html>, <head>, <title>, <body>, <h1>, <p>, <ul>, <li>, and <a>.

## Desired Results

In order for this project to be a success I will have a valid index.html, all the required semantic tags, a working GitHub link, and a .zip file that will contain my scope and index.

## Exclusions

My project cannot contain CSS styling, JavaScript, images, or external framework. This is a simple, clean html document.
